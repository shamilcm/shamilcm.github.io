about me

I've graduated from National Institute of Technology, Calicut (India) in June 2013, with major in Computer Science and Engineering. I'm interested in data mining, NLP, machine learning, web technologies and design. I have done projects in various areas of computer science including computer security, operating systems, cloud computing, AI, multimedia and web.

You can find details about the work I've done from this website.

projects

Experimental Operating System

 Design and implementation of an instructional operating system and underlying architecture with basic features like multiprogramming and filesystem as part of developing operating systems laboratory course work for junior year undergraduate computer science students.

StegoBots in Google Plus

Implemented a trojan horse which steals saved passwords from Google Chrome, encodes them in images and spreads through Google Plus. The passwords are retrieved using a Google Plus / Picasa Application, which scans shared pictures for stolen passwords.

HPC and Virtualization in the Cloud

Set up a scalable private cloud in the campus using open source tools like Eucalyptus and Rocks, capable of high performance computing and virtualization.

Stock Market Simulator (Bulls n' Bears)

Developed and conducted a web based stock market simulation game based on live quotes from NSE(India) using PHP and MySQL as part of technical fest, Tathva for two successive years. The game had a participation of more than 2000 users.
